Fixed: I-90 Form (Section 4 Disability Question)

Junaid Khan March 11, 2026

We fixed an issue in the Green Card Replacement (Form I-90) where the disability question in Section 4 could still prompt users for additional information even when “No” was selected for having a disability. This behavior caused confusion because the follow-up field should only appear when the user selects “Yes.”

The issue was reported by a user on March 8, and the form logic has now been corrected so that the follow-up question only appears when appropriate. Thanks to the user who reported this. These reports help me improve the reliability of Fillvisa forms.
